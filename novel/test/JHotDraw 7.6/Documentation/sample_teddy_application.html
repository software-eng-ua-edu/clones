JHotDraw 7: Teddy Sample Application

Teddy is a tiny text editor, which has been built using the JHotDraw application framework.

Launch JHotDraw Teddy:

If Teddy is launched on Mac OS X, it uses the Mac OS X document interface. Teddy aims to implement all Apple Human Interface Guidelines which are appropriate for a text editor.

If Teddy is launched on other platforms, it uses the single document interface (SDI), aiming to be a good citizen on Windows, Linux and Unix operating systems.
