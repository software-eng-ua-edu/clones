JHotDraw 7 Tutorial: Purpose

What can I do with JHotDraw?

JHotDraw 7 is designed for use in the following scenarios:

  • You want to integrate a drawing editor with basic SVG support into your application.
  • You want to create a drawing editor which supports a custom set of figures.
  • You want to create a drawing editor which provides a custom means of interaction.
  • You want to create a diagram editor of some kind.
  • You want to create a graphical editor for a system consisting of elements with behaviors and relationships.
  • You want to implement a document oriented application.
